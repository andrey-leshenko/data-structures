# data-structures
Basic data structures implemented in C# in the simplest way possible.

## Binary Heap
http://en.wikipedia.org/wiki/Binary_heap

| Operation     | Complexity        |
| ------------- |:----------------- |
| find max      | O(1)              |
| delete max    | O(log n)          |
| insert        | O(log n)          |
| construct     | O(n)              |
| merge         | O(m * log(n + m)) |
